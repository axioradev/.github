Axiora provides structured, XBRL-parsed financial data for every listed company in Japan. One API for financials, ownership trajectories, capital allocation, board composition, and more — sourced directly from official EDINET filings.

### What we cover

- **Financial statements** — revenue, income, assets, cash flows, EPS, ROE for 4,125 companies
- **Ownership intelligence** — large shareholding trajectories, accumulation signals, cross-holdings
- **Capital allocation** — FCF deployment classification, buyback execution tracking
- **Board composition** — directors, independence, gender breakdown
- **Voting results** — shareholder meeting proposal outcomes

### Built for developers

- REST API with 30-minute data freshness
- Python SDK (`pip install axiora`)
- MCP server for Claude, Cursor, and AI agents
- Bulk CSV export for quant workflows
- Webhook alerts for new filings
